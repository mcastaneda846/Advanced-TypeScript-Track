# Ecommerce Lite - React Project with TypeScript

The project was created with [Create React App](https://github.com/facebook/create-react-app) and is set up to use **TypeScript**.  

It includes:  
- Defined interfaces for **Product**, **User**, **Address**, and **Dimensions**.  
- Mock data for **products** and **users** in a centralized file (`data.ts`).  
- A list of products rendered on the main screen using typed components.

---

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in development mode.  
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.  

The page will reload if you make edits and you will also see any lint errors in the console.

---

### `npm test`

Launches the test runner in interactive watch mode.  
See the [running tests](https://facebook.github.io/create-react-app/docs/running-tests) section for more information.

---

### `npm run build`

Builds the app for production in the `build` folder.  
It correctly bundles React for production and optimizes the build for best performance.  

See the [deployment](https://facebook.github.io/create-react-app/docs/deployment) section for more information.

---

### `npm run eject`

**Note: this is a one-way operation.**  

If you are not satisfied with the default build configuration, you can `eject` to take full control of the scripts and dependencies (webpack, Babel, ESLint, etc.).

## Project Structure

- `/components` → Reusable components like `ProductCard` and `UserCard`.  
- `/interfaces` → Definitions of `Product`, `User`, `Address`, `Dimensions`.  
- `/data.ts` → Mock data of products and users.  

---

## How to Run

1. Clone the repository.  
2. Run `npm install` to install dependencies.  
3. Run `npm start` to start the app in development mode.  
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
