# Evidence - Module 5 TypeScript

This repository contains all the evidence corresponding to the user stories developed during **Module 5 of the TypeScript course**.  

The goal is to keep an organized record of each user story, making it easy for anyone to review the progress and results of the course.

## Current projects and evidence

### Ecommerce Lite
- Project built with **React and TypeScript**.
- Defined interfaces for **Product**, **User**, **Address**, and **Dimensions**.
- Mock data for products and users in a centralized file (`data.ts`).
- Product listing rendered on the main screen using typed components.
- Serves as evidence for user stories related to types, interfaces, and basic rendering.
